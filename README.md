
# Project Title

Admin and User Registration


## Run Locally

-Clone the project
-Go to the project directory

-Install dependencies
```bash
  npm install
```

Start the server
```bash
  npm start
```

## Features

- Sign and Signup
- Add ,update and view Admin
- Add,update and view User

## Tech Stack

**JavaScript,NodeJS,Expressjs,Mysql,Sequelize**
